# Perceptron com 50 Pesos (Excel)

Este projeto consiste na implementação de um **Perceptron com 50 pesos**, desenvolvido como trabalho acadêmico.

O modelo foi construído utilizando **Microsoft Excel**, onde toda a lógica do perceptron foi estruturada por meio de fórmulas na planilha.

## 📊 Descrição

O perceptron é um dos modelos mais simples de **redes neurais artificiais**, utilizado para tarefas de **classificação binária**.

Neste projeto:

* O modelo possui **50 pesos de entrada**
* As operações matemáticas são realizadas diretamente em células do Excel
* A planilha simula o funcionamento de um perceptron tradicional

## 🧠 Funcionamento

O perceptron recebe múltiplas entradas, cada uma associada a um peso.

O cálculo básico segue a lógica:

1. Multiplicação das entradas pelos pesos
2. Soma ponderada dos valores
3. Aplicação de uma função de ativação para gerar a saída

Toda essa estrutura foi implementada utilizando **fórmulas no Excel**.
